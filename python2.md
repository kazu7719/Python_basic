## pythonでの変数について
→Rubyと同様に、 変数名　= 格納する値

# piという変数の中身を、円周率の3.14159であると定義する
pi = 3.14159

# piの内容を表示させると、格納された3.14159が出力される
print(pi)
# →3.14159

# 何度実行しても、同じ結果が出力される
print(pi)
# →3.14159

# print関数
→Pythonに用意された、ターミナルに値を出力する関数

name = "Taro"

# 画面に出力する
print(name)

#　代入に関しては基本Rubyと同様


## 入力機能
→input関数を利用し、ユーザーが入力した値をコードないで使用する
(Rubyのgetと一緒)

# 入力機能が起動、入力された値が文字列として返る
name = input()

print(name)

## バックスラッシュ記法はRubyと同様


## 配列の生成

変数 = []

## 配列に値を追加する方法
→appendメソッドを使用

配列.append(追加する要素)

## 配列の値の取得
配列[添字]

変数 = 配列[添字]
→取得した要素は別の変数に代入することが可能

pencil_case = ["ペン", "消しゴム", "定規"]
pencil_case.append("えんぴつ")

print(pencil_case)
print(pencil_case[1])

##　配列の値の変更
配列[添字] = 値

pencil_case[1] = "修正ペン"

## 配列が持つ要素の数の数え方
→len(配列名)

print(len(pencil_case))
