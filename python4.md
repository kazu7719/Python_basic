## 繰り返し処理
→for文を用いる

for 変数 in 配列名:
    # 実行する処理

colors = ["あか", "あお", "きいろ"]

for color in colors:
    print(f"色: {color}")

## 回数を指定して繰り返す方法
→range関数を使う

for 変数 in range(繰り返したい回数):
    # 実行する処理

for i in range(10):
    print(i + 1)

※実行する前には、インデントが必要なことに注意


## 関数の定義方法

def関数名():
    # 実行する処理（処理の前にはインデント[半角スペース4つ分]が必須）

# 関数の定義
def say_hello():
    print("Hello World")

# 関数の実行
say_hello()


## 戻り値の指定方法
→return文を使う(Rubyと同様)

def sample():
    num = 2 * 3
    return num

print(sample())
# => 6

## 変数の値が使える範囲について(スコープ)

# グローバルスコープ
→関数の外側で定義した変数を意味する
範囲はファイル内全体で、使える範囲はグローバルスコープ内と、
ローカルスコープ内の両方

#　ローカルスコープ
→関数の内側で定義した変数を意味する
範囲はそれぞれの関数内のみで、使える範囲はローカル変数を定義した関数内のみ

# スコープを跨いだ変数の参照と変更
→ローカルスコープからグローバルスコープへの参照はできるが、変更は不可
グローバルスコープからローカルスコープへの参照・変更は不可なので、
return文で返す必要あり

## 引数について

def square(number):  # ②仮引数numberで、実引数「3」の値を受け取る
    print(number * number)  # ③「3」が代入されたnumberを使って「3 * 3」を行い出力する

square(3)  # ①数値「3」を実引数にして関数を呼び出し
