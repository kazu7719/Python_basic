## 辞書について

変数(辞書名) = { キー1 : バリュー1, キー2 : バリュー2, キー3 : バリュー3 }

student = {"name":  "John", "age": 10 }
teacher = {"name": "Mike", "age": 25 }

print(student)
print(teacher)

## 辞書に値を追加する方法

辞書名[追加するキー] = 値

teacher["subject"] = "English"

## 辞書の値の取得方法

辞書名[取得したい値のキー]

print(teacher["name"])

## 辞書の値の変更方法

辞書名[変更したい値のキー] = 値

teacher["name"] = "Emma"

## RubyとPythonのブロック終了の表現方法

# Ruby
→endでブロックし終了を表現

if 条件式

 処理①
 処理②

end
→endでブロック終了

# Python
→インデントの文字数の変化でブロック終了を表現

if(条件式):
    処理①
    処理②

print("文字")
→インデントの文字数が4から0に変化してるのでブロック終了
→1つのインデントは、半角スペース4つを入れることが推奨

ネスト(処理の中にさらに処理を入れ入れ子にする)させるには、
もう一つインデント、半角スペースを8つ入れる

if(条件式):
    処理①
    if(条件式):
        処理②
